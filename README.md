# Hostel-Management
Modelled on our college hostel.

This is meant to help both employees and residents at the hostel

Its functions are:
<br>1. Admission
<br>2. Complaint management
<br>3. Visitor Entry/Exit
<br>4. Night out
<br>5. Mess Registration
